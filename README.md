# ADA Final Project (Smoking vs Frequent Mental Distress)
 
## Project Description
This repository includes the cleaned BRFSS dataset for ADA Final project and R code used to analyze it. The project aims to assess the association between smoking and frequent mental distress among U.S. adults in 2022, using the BRFSS Data.

## Files Included
- `brfss_clean_ada.csv`: Cleaned version of the BRFSS 2022 dataset
- `ada_analysis_code.Rmd`: R Markdown used to summarize and visualize the data
- `README.md`: This file
 
## What the Code Does
- Reads in the BRFSS dataset
- Performs basic data cleaning (e.g., renaming columns, filtering)
- Creates summary statistics (e.g., age, sex, race)
- Generates plots to visualize the data
 
## How to Run the Code
1. Download or clone this repository to your computer
2. Open `ada_analysis_code.Rmd` in RStudio
3. Make sure your working directory is set to the folder where the files are saved
4. Run the script
